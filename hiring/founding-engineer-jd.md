# Job Description: Founding Engineer

## Role Overview
We are looking for our first Founding Engineer to lead the technical implementation of the WalkingPad Desktop application. You will work closely with the CTO and UX Designer to build a world-class desktop experience for WalkingPad users.

## Responsibilities
- Lead the implementation of the macOS application using Swift and SwiftUI.
- Optimize the Bluetooth Low Energy (BLE) integration for reliable treadmill control.
- Implement the "Liquid Glass" UI components defined in the design system.
- Integrate with 3rd party services like Notion and Strava.
- Build the "Stats Overlay" and "Global Hotkeys" features.

## Requirements
- Strong experience with Swift, SwiftUI, and macOS development.
- Experience with CoreBluetooth and BLE protocols.
- Familiarity with Notion API and OAuth2 flows.
- Ability to work autonomously and lead technical decisions.
- Passion for health, fitness, and productivity tools.
