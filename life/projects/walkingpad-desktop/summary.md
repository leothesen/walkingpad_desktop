# WalkingPad Desktop

Native macOS menu-bar app for controlling and monitoring WalkingPad treadmills over Bluetooth.

## Goals
- Understand the existing codebase and architecture.
- Implement new features and fix bugs as requested.

## Status
- Core controls and reliability improved (WAL-7): 0.1 km/h increments, debouncing, and state-aware start/stop logic implemented.
- Initial research completed (WAL-5).
- Architecture and core services identified.
- Automated testing infrastructure setup in progress (WAL-11).
